# FirstAid.AI – Intelligent First Aid Assistant

FirstAid.AI is an AI-powered emergency assistant designed to help identify injuries and provide first aid guidance using image analysis.

The system uses AI image recognition to analyze injuries and provide immediate first aid recommendations.

AI-powered First Aid Assistant web app built with React and Vite. Provides injury guidance via AI directly from the frontend. Backend integration for storing emergency cases are planned for future versions.

## Features

AI Injury Detection
Upload or capture an image of an injury and receive AI-powered analysis.

First Aid Guide
Provides first aid instructions for common injuries.

Hospital Finder
Uses GPS to locate nearby hospitals during emergencies.

User Authentication
Secure login and sign-up system.

Image Upload Options

* Camera capture
* File explorer upload

Helper Injury Detection
Images named after injuries (example: burns.jpg) can trigger helper hints.

## How to Test

1. Login or sign up
2. Click "Injury Detection"
3. Upload or capture an image
4. Click "Analyze" to get AI injury analysis

### Sample Injury Images for Testing

To easily test the AI system, create a folder called `sample-injuries` in the project root with the following example images:

- cuts.png
- burns.jpg
- sprain.jpg
- fracture.jpg
- snake-bite.png

These images can be used by hackathon judges to quickly test the AI injury detection feature. Upload them via the "Upload from File Explorer" option to see the AI analysis in action.

This project was built for a hackathon to demonstrate how AI can assist with emergency first aid guidance in real-world situations.

FirstAid.AI aims to make emergency assistance more accessible through intelligent injury detection and instant first aid recommendations.
